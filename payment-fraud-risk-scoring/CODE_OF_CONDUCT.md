# Code of Conduct

## Our Standards

- Be respectful and constructive.
- Focus feedback on code and behavior, not people.
- Assume good intent and communicate clearly.
- Avoid harassment, discrimination, or exclusionary language.

## Enforcement

Project maintainers may remove comments, commits, or contributions that violate
this code of conduct, and may restrict participation for repeated violations.

## Reporting

Report unacceptable behavior privately to project maintainers.
